## Dotfiles

- Neovim
- Vim
- IdeaVim
- Sioyek (PDF reader)
- Karabiner (key remap)
- Aerospace (vm)
- Neovide as the default app for text and source files

### Installation

```bash
## needs an update and some manual work
cd dotfiles
bash first_init.sh ~/dotfiles
```

### Default text app

```bash
bash scripts/set-default-text-app.sh
```

### Deinit

```bash
cd dotfiles
bash deinit.sh
```
